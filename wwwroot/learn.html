Benefits of Cloud Computing

Benefits include: ability to scale quickly, server service free maintenaince, flexible pay models, always available services, etc.

Risks of Cloud Computing

The risks with cloud computing is trust. Trust is estowed upon the service provider by the user. The user expects the service provider to provide a reliable service.

Moving to a cloud computing system can also be difficult depending on the size, number of applications being used, and if the applications are easily hostable.
