What is Cloud Hosting?

Cloud hosting is using a network of virtual and physical servers, that the user does not control, to run software solution and provide services or information to consumers on the internet.

Some key features include being able to scale your solution quickly and efficiently, you pay for the resources you use, automation will keep your solution's hosting software up-to-date.
